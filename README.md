# SearchInDouBan
查豆瓣 —— 通过复制、分享，实现快速简易查询豆瓣内容的 Android APP。

![demo.gif](https://i.loli.net/2019/02/11/5c616d00724f1.gif)



## 特性

- 从【长按复制】菜单快速搜索、从分享文字到本应用进行搜索和直接进入程序主界面搜索

- 支持在本地豆瓣 APP 打开当前结果页
- 分享页面链接或在浏览器打开
- 广告过滤
